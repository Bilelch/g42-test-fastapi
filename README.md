# g42-test-fastAPI

## Getting started

### Technology stack

- FastAPI
- Pandas
- Pyhton >= 3.6

#### Backend (FastAPI)

```sh

# Create main folder for development
mkdir code

# Go into folder
cd code

#Create virtualenv
pip install virtualenv
virtualenv venv1
venv1/scripts/activate

# Checkout shared code
git clone git@gitlab.com:Bilelch/g42-test-fastapi.git

# Goto site project
cd g42-test-fastapi

#Install requirements
pip3 install -r requirements.txt

# Run app
run.sh

```
